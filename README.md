# grace's schedule.

Sebuah *website* statis sederhana yang menampilkan jadwal perkuliahan mingguan. Proyek ini dibangun sebagai pemenuhan tugas praktikum (Lab) mata kuliah Pemrograman Web.

## Tentang Proyek
Website ini dirancang untuk menyajikan informasi jadwal kelas akademik secara terstruktur dan mudah dibaca. Melalui tugas ini, saya mempraktikkan langsung cara menyusun data menggunakan elemen *Table* di HTML, serta mengatur *layout* dan *styling* antarmuka menggunakan CSS agar terlihat lebih modern dan interaktif.

## Fitur Utama
*   **Hero Section:** Halaman penyambutan (*Welcome to Grace's Schedule*) dengan elemen visual yang estetik.
*   **Tabel Jadwal Terstruktur:** Menampilkan daftar mata kuliah semester 3, lengkap dengan informasi hari dan waktu pelaksanaan.
*   **Social Media Navigation:** Terdapat ikon tautan langsung ke profil Instagram dan LinkedIn yang terintegrasi di bagian *header*.

## Tech Stack
Proyek ini dibangun menggunakan teknologi dasar *web development*:
*   **HTML5:** Untuk membangun kerangka *website* dan menyusun baris serta kolom pada tabel jadwal.
*   **CSS3:** Untuk memberikan *styling* pada tabel (seperti warna *header pink*, garis batas tabel, tipografi) dan memastikan tata letaknya rapi.
